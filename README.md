#2D Crazy Bird Game
##Short Introduction
	This is a 2D ANGRY Bird–style physics game built with Unity.
	The player launches a bird using a slingshot to destroy enemies or break obstacles.
	The game uses realistic physics to simulate trajectory, impact, and object destruction.
##Game Objective
1. Use the slingshot to launch the bird toward the enemy structures.
2. Destroy all enemies to complete the level.
3. Choose between two seasons (Default or Winter) before playing.
4. The game ends when all targets are eliminated—or when the player runs out of birds.
##Game Rules
1. Press Start to enter the Season Selection screen.
2. Choose between Fall or Winter levels.
3. Click and drag the bird to pull the slingshot.
4. Release to launch the bird toward the obstacles.
5. Destroy all enemies on the map to win the level.
6. If you miss and run out of birds, you lose.
7. Players can pause the game using the Esc key.
